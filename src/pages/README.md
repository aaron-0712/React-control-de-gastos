# Pages

Son las vistas completas de la aplicación (Login, Dashboard, Reportes).

- No contienen lógica compleja de estilos (eso va en `components`).

- Su función principal es **orquestar**: llaman a los `hooks` de `querys` para traer datos y los pasan a los `templates`.

- Cada archivo aquí suele corresponder a una ruta en el navegador.
